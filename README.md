# Install

# Clone or Download this repository

Don't be stupid for clone github repository

# Run

`composer install`

Copy .env.example file to .env file and fill dbname, dbuser, dbpass

`php artisan migrate --seed`

# Run local server

`php artisan serve`

Open in your browser
